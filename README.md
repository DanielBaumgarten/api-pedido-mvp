# API Pedidos MVP

API REST desenvolvida em Node.js e Express para gerenciamento de pedidos de uma empresa de vendas de produtos.

## Objetivo

Este projeto foi desenvolvido como trabalho avaliativo com o objetivo de aplicar os conceitos de:

- APIs RESTful
- Validações e regras de negócio
- Arquitetura em camadas
- Versionamento com Git/Github
- Testes de endpoints utilizando Insomnia

---

## Tecnologias Utilizadas

- Node.js
- Express
- Nodemon
- Git
- Github
- Insomnia
